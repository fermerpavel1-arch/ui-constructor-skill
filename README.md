# ui-constructor

Grok / AI-agent **skill**: curated **Magic UI** + **React Bits** kit, anti-AI-slop rules, MCP install paths, and an industrial portfolio reference stack.

## Install

Copy into your agent skills directory, for example:

```text
~/.grok/skills/ui-constructor/
```

Keep structure:

```text
ui-constructor/
  SKILL.md
  references/
    anti-slop.md
    library.md
    mcp.md
    poc-patterns.md
```

Optional related skills (not bundled): `design-knowledge`, `frontend-design`, `site-user-debug`.

## What it does

- Prefers a curated component library (tiers ★★★ / ★★ / ★)
- Blocks common AI “slop” UI patterns
- Documents Magic UI MCP + CLI fallback
- Documents React Bits manual PoC flow
- Locks architecture lessons: glass chrome vs solid cards, one WebGL bg, nav outside transitions

## Privacy

This public package is **sanitized**: no personal names, local paths, portfolio URLs, emails, or credentials.
